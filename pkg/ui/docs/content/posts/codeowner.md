# CODEOWNER

File for description of command responsibility areas

### Example from GitLab

![gitlab-codeowner](./images/codeowner.png)

### Structure

```
- groupCompany
  - project-workspace-1
    - project-1
    - project-2
    - project-3
  - project-workspace-N
    - project-1
    - project-N
  - maintainers
    - frontend
    - backend
    - database
    - etc...
```
